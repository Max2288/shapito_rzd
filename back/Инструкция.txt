Файл main_max.py отвечает за генерацию таблицы под названием new.xlsx , в которой лежат данные в формате профессия, компетенция, индикаторы, курс, описание
Файл pars.py отвечает за генерацию словаря формата {профессия: {компетенция: {индикатор: [[курс, описание]...]}}} 
Файл Новая таблица(1).xlsx - таблица, в которой указаны профессия компетенция и индикаторы
Файл Courses.xlsx - таблица, в которой указаны описание курсов и с которой считаны название курсов 
Файл Цифровые компетенции.xlsx - итоговая таблица, которой может пользоваться руководство 
Файл Сравнения.xlsx - таблица, в которой представленны сравнения комманд
Файл Смета.xlsx - смета 
Файл Источники.txt - источники, с которых брали информацию 
Файл Шапито.pptx - презентация 
Файл Работа_сайта.mkv - видео показ работы сайта 
Файл Сайт.docx - переходник на сайт
